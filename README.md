# Number Guessing Game

A simple and interactive Java-based Number Guessing Game where players guess a randomly generated number. The game is accessible both via the command line and a web browser. This application demonstrates the use of Jetty for lightweight server capabilities and is packaged as a deployable artifact.

---

## Features

- Play the game via:
  - **Command Line**: Direct interaction with the program.
  - **Web Browser**: Accessible through an HTML interface.
- Random number generation to ensure fair gameplay.
- Feedback for guesses (`Too High`, `Too Low`, or `Correct`).
- Tracks the number of attempts.
